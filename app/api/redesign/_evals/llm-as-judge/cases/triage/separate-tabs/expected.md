## AOI 1

Completed tasks move to a separate "Completed" tab instead of staying visible inline, so users have to switch tabs to confirm a check-off registered, then switch back to continue with active tasks — repeated back-and-forth navigation just to verify actions are taking effect.

## AOI 2

Tasks in the Completed tab can't be unchecked — there's no way to move a task back to the active list if it was checked off by mistake or needs to be reopened, leaving users stuck with no path to reverse the action.
